Ce dépôt regroupe l’ensemble des **labs MLOps** réalisés dans le cadre du module, couvrant tout le cycle de vie d’un projet Machine Learning 

## Labs réalisés

### Lab 1 : Du notebook au mini-système production-ready

### Lab 2 : Code Source Management (Git)

### Lab 3 : Versionnement des données et pipelines ML avec DVC

### Lab 4 : Mise en place d’un pipeline CI/CD pour un projet ML

### Lab 5 : Déploiement conteneurisé d’un modèle ML

Chaque dossier `labX` contient les captures d’écran nécessaires pour justifier l’exécution et la validation du lab correspondant.
