## Lab2 : Code Source management

### Objectif
Apprendre à versionner correctement un projet Machine Learning avec Git.

### Points clés retenus
- Initialisation d’un dépôt Git
- Compréhension du cycle `working tree → staging → commit`
- Bonnes pratiques de messages de commit
- Utilisation de `.gitignore` pour exclure les fichiers non versionnables
- Distinction entre code, configuration et artefacts
